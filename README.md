# MyCar.kz Manual Testing Project

## Project Overview

This project contains manual functional testing of the **"Used Cars"** section on the MyCar.kz website.

The goal of the project was to verify the correct behavior of the main user flows including search, filters, sorting, car details page, and credit calculator.

---

## Scope of Testing

The following functionality was tested:

- Navigation and pagination  
- Sorting functionality  
- Quick filters  
- Advanced filters  
- Car card interactions  
- Car details page  
- Credit calculator  

Out of Scope:

- API testing  
- Database validation  
- Performance testing  
- Security testing  
- Mobile version  

---

## Testing Types Performed

- Functional Testing  
- Negative Testing  
- Boundary Value Testing  
- Smoke Testing
